-   テーマの変更

    `$PROFILE` 内の `paradox.omp.json`のファイル名を変更

## json customize

-   基本的に oh-my-posh [公式ドキュメント](https://ohmyposh.dev/docs) の`🌟Segments`を見ることでカスタマイズが可能

-   `Get-PoshThemes` でテーマの一覧が表示される
